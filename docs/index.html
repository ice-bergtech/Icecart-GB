Another ROM Cartridge for the Gameboy


Goals

  • STM32 Processor
  • Support All MBC types (including TPP1)
  • 3.8mm Mic/Speaker jack
  • 2x8 GPIO pins
  • 32MBit Memory
  • MicroSD Card slot
  • Support FreeRTOS
