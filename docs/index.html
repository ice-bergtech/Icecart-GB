Icecart-GB

-A Complete ROM Cartridge for the 8-bit Gameboy-


Goals:

  • STM32 Processor
  • Simulate ALL MBC types, including TPP1
  • 32MBit Memory
  • MicroSD Card slot
  • 3.8mm Mic/Speaker jack
  • 2x8 GPIO pins
  • Support FreeRTOS
